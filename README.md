# My voice
 10-seconds-a-day audio journal with daily notifications, quotations, and tags.

[Download](https://play.google.com/store/apps/details?id=com.tamnada.audio_journal&hl=en&gl=US) on Google Play Store :robot:

## Tech stack
Flutter + Dart
SQLite database
